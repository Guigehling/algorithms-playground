# Algorithms Playground

A simple repository to study and experiment with algorithms.

Each algorithm lives in its own folder and can have implementations in different programming languages. Every implementation is independent and can be run and tested separately.

## Structure

```
algorithms-playground/
├── fenwick-tree/
│ ├── java/
│ ├── kotlin/
└── scripts/
```

## Running

Run each implementation from its own folder.

## Tests

Tests are defined per language.

## Adding a new algorithm

1. Create a new folder
2. Add one or more language implementations
3. Add code and tests
4. Ensure it runs independently
